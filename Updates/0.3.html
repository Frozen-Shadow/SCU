Version 0.3.378

春节假期够长，所以更新了不少东西。注意使用前备份好您的输入法配置目录 ~/Library/Rime（如果不知道在哪里，可以从 Squirrel 鼠鬚管输入法菜单里选择 Settings/用户设定，然后备份打开窗口里所有文件）。另外升级到 Squirrel 鼠鬚管输入法最新版本（目前是 0.9.17）也能避免一些可能的麻烦。这个版本之后大体功能框架就完整了，接下去我会逐步增加一些针对特定输入法方案的定制功能，如果有功能需求请让我知道。

新增功能：

  • Apps 设定：在特定应用程序中开启和关闭强制 ASCII 输入模式和软光标模式。
  • Schemata 设定：启用/停用 Squirrel 鼠鬚管输入法支持的各种输入方案。
  • Genaral 设定：增加方案选单标题和唤出快捷键设置。
  • 支持 Squirrel 鼠鬚管输入法新引入的配置项 show_notifications_via_notification_center（通过 OS X 10.8+ 的通知中心显示输入法切换通告，General 页）和 line_spacing（垂直候选条的行距，Style 页）。
  • 增加 Ask Squirrel to Deploy 功能（主菜单 File 或者 主界面 Help 页）。
  • 增加了简体中文本地化。

问题修复：

  • 现在能够正确处理原先手工编辑配置文件自定义的配色方案（注意颜色值用单角双引号括起来，或者使用完整 ABGR 编码，最开始的 0xff 不可省略）。

系统要求：

  • Mac OS X 10.7 或以上版本
  • Squirrel（鼠鬚管）输入法 0.9.13 或以上版本

Version 0.2

开源中文输入法 Squirrel（鼠鬚管） 终于有了一个图形化的配置工具。目前还只支持最基本的配置选项，但总体框架已经完整，以后会增加更多配置选项的支持。SCU 本身也在 GitHub 开源，有兴趣的朋友可以一起来改进和增强它，如果有任何使用上的问题也可以在 GitHub 上提交错误报告（见下面的相关链接）。

已支持的选项：

  • 基本选项：缺省键盘布局、开关切换输入状态时的系统提示；
  • 候选条选项：垂直/水平方向、候选词数、字体、透明度、颜色方案、圆角大小等。

尚未支持的选项：

  • 应用选项：在指定应用中自动强制 Squirrel 进入 ASCII 字符输入模式；
  • 输入方案特定选项：开关和定制特定输入方案；
  • 其他官方定制指南中描述的配置选项。
